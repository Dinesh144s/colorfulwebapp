# 🌈 Colorful WebApp

A simple and vibrant Java web application built using **Maven** and deployed on **Apache Tomcat**. Perfect for learning full-stack deployment in a DevOps or Java development environment.

---

## 💡 Features

- 🎨 Beautiful, colorful HTML interface
- ⚙️ Built with Maven (`.war` file generation)
- 🚀 Deployable on Apache Tomcat (tested on EC2)
- 🐧 Runs great on Ubuntu/Linux
- ☁️ Ready for cloud & CI/CD integration (GitHub Actions, AWS, etc.)

---

## 🛠️ Technologies Used

- Java (Servlet/JSP)
- Maven
- Apache Tomcat
- HTML/CSS
- Git & GitHub

---

## 📦 How to Build

```bash
# Clone the repo
git clone https://github.com/Dinesh144s/colorfulwebapp.git
